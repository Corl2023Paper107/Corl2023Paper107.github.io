Semantic Map Failure Modes

False Positive Goal [47%]

The semantic segmentation model incorrectly labels the counter as "bed" and stop at it. This can be seen in the top-down semantic map where a large blue blob represents the incorrectly-labeled goal.

Planning Failure [23%]

The agent gets stuck on scene geometry that the planner does not see in its map.

Stops Too Early [9%]

The agent explores the environment, and at some point calls a 'STOP' action too early.

Exploration Failure [9%]

The agent explores the environment, but not fully and does not find the goal.

Failed to Find Goal [6%]

The agent passes by the potted plant multiple times but fails to recognize it.

Spilling of Semantic Segmentation Mask [6%]

The semantic mask spills onto the wall, causing the wall to be labeled as couch. The agent then navigates and stops at the wall.
