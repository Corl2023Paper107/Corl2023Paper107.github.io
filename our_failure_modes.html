Our Failure Cases

Missing Annotations

(Object Goal = TV) The agent calls the stop action right next to goal, but due to missing annotations, the agent is marked as "failed".

Exploration in Loops

(Object Goal = TV) The agent repeatively explores the same area that it has already seen before.

Missed Staircases

(Object Goal = TV) The agent needs to go to the downstairs living room in order to find the goal. However, it fails to recognize the stairs in the hallway.

Recognition Errors

(Object Goal = Toilet) The agent repeatively visits the bathroom, but the semantic segmentation model fails to recognize the toilet.

Cannot Avoid Broken Floor Geometry

(Object Goal = Toilet) The agent attempts to navigate to the goal, but the floor mesh to the toilet goal is incomplete, and the agent fails to get near the goal.

Stuck on Stairs

(Object Goal = TV) The agent repeatively tries to go up/down the stairs, but these are cosmetic and don't lead to new floors.

Switching Error

(Object Goal = TV) The agent at multiple points gets fairly close to the goal, but not close enough. The exploration policy then diverges. (Around ~23 seconds in)

Stop Right Away

(Object Goal = TV) The agent immediately stops due to sampling from a distribution of actions.
